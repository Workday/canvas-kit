# Canvas Kit Text Area

Text Areas allow users to enter and edit multiple lines of text.

Undocumented props are spread to the underlying `<textarea>` element.

[> Workday Design Reference](https://design.workday.com/components/inputs/text-area)

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Basic Example

Text Area should be used in tandem with
[Form Field](/docs/components-inputs-form-field-react--default) to meet accessibility standards.

<ExampleCodeBlock code={Default} />

## Disabled

Set the `disabled` prop of the Text Area to prevent users from interacting with it.

<ExampleCodeBlock code={Disabled} />

## Placeholder

Set the `placeholder` prop of the Text Input to display a sample of its expected format or value
before a value has been provided.

<ExampleCodeBlock code={Placeholder} />

## Label Position

Set the `labelPosition` prop of the wrapping Form Field to designate the position of the label
relative to the Text Area. `labelPosition` accepts the following values:

- `FormField.LabelPosition.Top` (Default)
- `FormField.LabelPosition.Left`

<ExampleCodeBlock code={LabelPosition} />

## Required

Set the `required` prop of the wrapping Form Field to `true` to indicate that it's required. Labels
for required fields are suffixed by a red asterisk.

<ExampleCodeBlock code={Required} />

## Grow

Set the `grow` prop of the wrapping Form Field to `true` to configure the Text Area to expand to the
width of its container.

<ExampleCodeBlock code={Grow} />

The `grow` prop may also be applied directly to the Text Area if Form Field is not being used.

## Resize Constraints

Set the `resize` prop of the Text Area to restrict resizing of it to certain dimensions. `resize`
accepts the following values:

- `TextArea.ResizeDirection.Both` (Default)
- `TextArea.ResizeDirection.Horizontal`
- `TextArea.ResizeDirection.None`
- `TextArea.ResizeDirection.Vertical`

<ExampleCodeBlock code={Resize} />

## Ref Forwarding

Text Area supports [ref forwarding](https://reactjs.org/docs/forwarding-refs.html). It will forward
`ref` to its underlying `<textarea>` element.

<ExampleCodeBlock code={RefForwarding} />

## Error States

### Alert

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Alert` and provide an
explanation using the `hintText` prop to set the Text Area to the Alert state.

<ExampleCodeBlock code={Alert} />

The `error` prop may also be applied directly to the Text Area with a value of
`TextArea.ErrorType.Alert` if Form Field is not being used.

### Error

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Error` and provide an
explanation using the `hintText` prop to set the Text Area to the Error state.

<ExampleCodeBlock code={Error} />

The `error` prop may also be applied directly to the Text Area with a value of
`TextArea.ErrorType.Error` if Form Field is not being used.
