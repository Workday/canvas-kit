# Canvas Kit Checkbox

Checkboxes allow a user to select zero, one, or multiple values from a predefined list of 7 or less
options.

Undocumented props are spread to the underlying `<input type="checkbox">` element.

[> Workday Design Reference](https://design.workday.com/components/inputs/checkboxes)

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Default Example

Checkbox should be used in tandem with Form Field to meet accessibility standards.

<ExampleCodeBlock code={Default} />

## Disabled

Set the `disabled` prop of the Checkbox to prevent users from interacting with it.

<ExampleCodeBlock code={Disabled} />

## Indeterminate

Set the `indeterminate` prop of the Checkbox to `true` to indicate the Checkbox is neither checked
nor unchecked.

A common use case for an indeterminate Checkbox is when the value of a parent Checkbox is dependent
on a number of child Checkboxes. The parent Checkbox is set to the indeterminate state if some (but
not all) of its children are checked.

<ExampleCodeBlock code={Indeterminate} />

## Label Position

Set the `labelPosition` prop of the wrapping Form Field to designate the position of the label
relative to the Checkbox. `labelPosition` accepts the following values:

- `FormField.LabelPosition.Top` (Default)
- `FormField.LabelPosition.Left`

<ExampleCodeBlock code={LabelPosition} />

## Required

Set the `required` prop of the wrapping Form Field to `true` to indicate that it's required. Labels
for required fields are suffixed by a red asterisk.

<ExampleCodeBlock code={Required} />

## Ref Forwarding

Checkbox supports [ref forwarding](https://reactjs.org/docs/forwarding-refs.html). It will forward
`ref` to its underlying `<input type="checkbox">` element.

## Error States

### Alert

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Alert` and provide an
explanation using the `hintText` prop to set the Checkbox to the Alert state.

<ExampleCodeBlock code={Alert} />

The Alert state may be applied directly to the Checkbox using the static `Checkbox.ErrorType.Alert`
value if Form Field is not being used.

<ExampleCodeBlock code={AlertStandalone} />

explanation using the `hintText` prop to set the Checkbox to the Error state.

<ExampleCodeBlock code={Error} />

The Error state may be applied directly to the Checkbox using the static `Checkbox.ErrorType.Error`
value if Form Field is not being used.
