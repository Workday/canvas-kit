# Canvas Kit Switch

Switch is a selection control that is used to switch between two potential states.

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Default Example

Switch should be used in tandem with Form Field to meet accessibility standards.

<ExampleCodeBlock code={Default} />

If necessary, Switch can also be used on its own.

<ExampleCodeBlock code={Standalone} />

## Disabled

Set the `disabled` prop of the Switch to prevent users from interacting with it.

<ExampleCodeBlock code={Disabled} />

## Label Position

Set the `labelPosition` prop of the wrapping Form Field to designate the position of the label
relative to the Switch. `labelPosition` accepts the following values:

- `FormField.LabelPosition.Top` (Default)
- `FormField.LabelPosition.Left`

<ExampleCodeBlock code={LabelPosition} />

## Ref Forwarding

Switch supports [ref forwarding](https://reactjs.org/docs/forwarding-refs.html). It will forward
`ref` to its underlying `<input type="checkbox">` element.

<ExampleCodeBlock code={RefForwarding} />

## Error States

### Alert

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Alert` and provide an
explanation using the `hintText` prop to set the Switch to the Alert state.

<ExampleCodeBlock code={Alert} />

The Alert state may be applied directly to the Switch using the static `Switch.ErrorType.Alert`
value if Form Field is not being used.

<ExampleCodeBlock code={AlertStandalone} />

### Error

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Error` and provide an
explanation using the `hintText` prop to set the Switch to the Error state.

<ExampleCodeBlock code={Error} />

The Error state may be applied directly to the Switch using the static `Switch.ErrorType.Error`
value if Form Field is not being used.

<ExampleCodeBlock code={ErrorStandalone} />
