# Canvas Kit Loading Animation

Loading Animations make users aware that content is currently being loaded, processing, or that
change will occur on the page.

[> Workday Design Reference](https://design.workday.com/components/indicators/loading-animation)

## Installation

```sh
yarn add @workday/canvas-kit-react
```
