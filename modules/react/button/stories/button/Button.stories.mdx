# Canvas Kit Button

Clickable button elements that extend the native `<button>` element with Canvas styling.

[> Workday Design Reference](https://design.workday.com/components/buttons/buttons)

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Usage

#### Size

Buttons can be three different sizes, `small`, `medium`, and `large` (except for the
`TertiaryButton` which does not have a `large` size).

<ExampleCodeBlock code={Size} />

#### Icon

All Buttons except for the `DeleteButton` support icons. By default, they are rendered on the left
of the label, but you can move them to the right with the `iconPosition` prop.

<ExampleCodeBlock code={Icon} />

#### Grow

Buttons will collapse to the width of their content by default. To expand to the width of their
container, use the `grow` prop.

<ExampleCodeBlock code={Grow} />
