# Canvas Kit Text Input

Text Inputs allow users to enter words or characters without styling.

Undocumented props are spread to the underlying `<input type="text">` element.

[> Workday Design Reference](https://design.workday.com/components/inputs/text-input)

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Default Example

Text Input should be used in tandem with Form Field to meet accessibility standards.

<ExampleCodeBlock code={Default} />

If necessary, Text Input can also be used on its own.

<ExampleCodeBlock code={Standalone} />

## Disabled

Set the `disabled` prop of the Text Input to prevent users from interacting with it.

<ExampleCodeBlock code={Disabled} />

## Placeholder

Set the `placeholder` prop of the Text Input to display a sample of its expected format or value
before a value has been provided.

<ExampleCodeBlock code={Placeholder} />

## Label Position

Set the `labelPosition` prop of the wrapping Form Field to designate the position of the label
relative to the Text Input. `labelPosition` accepts the following values:

- `FormField.LabelPosition.Top` (Default)
- `FormField.LabelPosition.Left`

<ExampleCodeBlock code={LabelPosition} />

## Required

Set the `required` prop of the wrapping Form Field to `true` to indicate that it's required. Labels
for required fields are suffixed by a red asterisk.

<ExampleCodeBlock code={Required} />

## Grow

Set the `grow` prop of the wrapping Form Field to `true` to configure the Text Input to expand to
the width of its container.

<ExampleCodeBlock code={Grow} />

The `grow` prop may also be applied directly to the Text Input if Form Field is not being used.

<ExampleCodeBlock code={GrowStandalone} />

## Ref Forwarding

Text Input supports [ref forwarding](https://reactjs.org/docs/forwarding-refs.html). It will forward
`ref` to its underlying `<input type="text">` element.

<ExampleCodeBlock code={RefForwarding} />

## Error States

### Alert

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Alert` and provide an
explanation using the `hintText` prop to set the Text Input to the Alert state.

<ExampleCodeBlock code={Alert} />

The Alert state may be applied directly to the Text Input using the static
`TextInput.ErrorType.Alert` value if Form Field is not being used.

<ExampleCodeBlock code={AlertStandalone} />

### Error

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Error` and provide an
explanation using the `hintText` prop to set the Text Input to the Error state.

<ExampleCodeBlock code={Error} />

The Error state may be applied directly to the Text Input using the static
`TextInput.ErrorType.Error` value if Form Field is not being used.

<ExampleCodeBlock code={ErrorStandalone} />
