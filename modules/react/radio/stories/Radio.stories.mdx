# Canvas Kit Radio

Radio Buttons allow a user to select one value from a predefined list of 7 or fewer options.

Undocumented props are spread to the underlying `<input type="radio">` element.

[> Workday Design Reference](https://design.workday.com/components/inputs/radio-buttons)

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Default Example

Radio Buttons are intended to be grouped together using a Radio Group.

Radio Group should be used in tandem with Form Field where the `useFieldset` prop is set to `true`
to meet accessibility standards.

<ExampleCodeBlock code={Default} />

## Disabled

Set the `disabled` prop of the Radio Button to prevent users from interacting with it.

<ExampleCodeBlock code={Disabled} />

## Label Position

Set the `labelPosition` prop of the wrapping Form Field to designate the position of the label
relative to the Radio Group. `labelPosition` accepts the following values:

- `FormField.LabelPosition.Top` (Default)
- `FormField.LabelPosition.Left`

<ExampleCodeBlock code={LabelPosition} />

## Required

Set the `required` prop of the wrapping Form Field to `true` to indicate that it's required. Labels
for required fields are suffixed by a red asterisk.

<ExampleCodeBlock code={Required} />

## Radio Buttons with No Values

The `value` prop may be omitted from Radio Buttons, in which case the `value` prop of the Radio
Group should be set using the zero-based index of the selected Radio Button.

<ExampleCodeBlock code={NoValue} />

## Ref Forwarding

Radio Button supports [ref forwarding](https://reactjs.org/docs/forwarding-refs.html). It will
forward `ref` to its underlying `<input type="radio">` element.

<ExampleCodeBlock code={RefForwarding} />

## Error States

### Alert

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Alert` and provide an
explanation using the `hintText` prop to set the Radio Group to the Alert state.

<ExampleCodeBlock code={Alert} />

### Error

Set the `error` prop of the wrapping Form Field to `FormField.ErrorType.Error` and provide an
explanation using the `hintText` prop to set the Radio Group to the Error state.

<ExampleCodeBlock code={Error} />
