# Canvas Kit Information Highlight

A container to call out important information on a page or a section of a page that the user should
be aware of.

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Usage

<ExampleCodeBlock code={Basic} />

### Variants

`InformationHighlight` has four variants and each `variant` has two types of `emphasis`.

Variants: `default`, `informational`, `caution`, `critical`

Emphasis: `low`, `high`

The variants have different icons and colors to convey severity and the emphasis changes the
background color to have a lower or higher contrast. If no `variant` or `emphasis` is selected the
layout will default to `Variant: default, Emphasis: low`.

The following examples will show the `low` and `high` emphasis of the severity variants. `low`
Emphasis will be shown first and should be used when there is other more important information on
the page. `high` Emphasis will be shown second, and should be used when the highlight is not
competing with other, more important information.

#### Default

The default variant is for general callouts that do not convey a specific severity level. See the
[Basic Example](#basic-example) above — it uses the default `variant` and `emphasis` with all parts
filled in.

#### Informational

The informational variant is for _nice to have_ information, such as related features or
opportunities.

<ExampleCodeBlock code={Informational} />

#### Caution

The caution variant is for _important to know_ information, such as the potential consequences of
specific actions.

<ExampleCodeBlock code={Caution} />

#### Critical

The critical variant is for _must know_ information that could otherwise cause failure if the user
is unaware.

### CTA Placement

Use `ctaPlacement` to control where the link renders relative to the heading and body.

- `bottom` (default): the link stacks below the heading and body in document order.
- `end`: the link renders beside the content, vertically centered and right-aligned.
