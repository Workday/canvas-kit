# Canvas Kit Avatar

A component showing a user's photo with a circular crop.

[> Workday Design Reference](https://canvas.workdaydesign.com/components/indicators/avatar)

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Usage

<SymbolDescription name="Avatar" fileName="/react/" />

The example below shows multiple instances of a `Avatar` with various configurations. Like many of
our components, Avatar accepts an `as` prop, which lets you change the underlying semantic element
to a `<div>`. This should be done with caution to ensure the best accessibility. Generally,
`<button>` elements should be used for actions.

### Variant and Size

Avatar defaults to using a "light" `variant` property. You can change the `variant` prop to "dark"
by including `variant="dark"`.

Avatar defaults to using a "medium" `size` property. You can change the `size` prop to various
sizing options. See Component API for more details.
