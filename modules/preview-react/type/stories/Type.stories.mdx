# Canvas Kit React Type

`Type` is a group of text components that provide a common, ergonomic API to render text with
simplified usage of Canvas Kit type token.

## Installation

```sh
yarn add @workday/canvas-kit-react
```

## Text

### Usage

`Text` is a basic text component to render modified text. `Text` component composes `Box` component
that allows to use all layout props, such as `display`, `margin`, `lineHeight` etc. It also has
their own props such as `level` and `size` that should be provided together, because they refer to
type token level and size, and `isTruncated` allows to make text truncated with ellipsis on its end.

By default `Type` component is a `span` element, but it's possible to override the element by
passing `as` prop.

```tsx
<Text as="p" fontSize={14} fontWeight="medium">
  Text here...
</Text>
```

## Label

`Label` is a `label` element build based on the top of `Text` component with added ability to add
pointer cursor or disabled style.

### Usage

```tsx
<Label hasPointerCursor disabled={true}>
  Text here...
</Label>
```

## Type Level Components

Type Level Components, `TypeSubtextLevel`, `TypeBodyLevel`, `TypeHeadingLevel` and `TypeTitleLevel`,
apply type token styles to text element. Instead of passing type level token you can use one of Type
Level Components, like in example below:

```tsx
// v7
<p css={{...type.levels.subtext.small}}>Small Subtext Text</p>

// v8
<TypeSubtextLevel size="small">Small Subtext Text</TypeSubtextLevel>
```
