Pills are small elements that represent/label an input or attribute

## Installation

```sh
yarn add @workday/canvas-kit-preview-react
```

## Usage

```tsx
import * as React from 'react';
import {Pill} from '@workday/canvas-kit-preview-react/pill';

<Pill />;
```
