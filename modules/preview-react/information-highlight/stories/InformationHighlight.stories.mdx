# Information Highlight

An `Information Highlight` draws user attention to important details
## Installation

```sh
yarn add @workday/canvas-kit-react
```


## Usage
`Information Highlight` should be used to call out important elements of the page. It has three emphases built in:
Informational, Caution, and Attention. The emphases have different icons and colors to convey severity.
If no emphasis is selected the layout will default to Informational.
