# Canvas Kit Information Highlight

An `Information Highlight` draws user attention to important details

## Installation

```sh
yarn add @workday/canvas-kit-preview-react
```

## Usage
`Information Highlight` should be used to call out important elements of the page. It has three variants and two emphases
built in.

Variants: Informational, Caution, Attention

Emphasis: Low, High

The variants have different icons and colors to convey severity, the emphasis changes the background color to have a
lower or higher contrast. If no variant or emphasis is selected the layout will default to Informational, Low.
