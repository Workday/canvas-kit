# Canvas Kit React Text

`Text` is a group of text components that provide a common, ergonomic API to render text with
simplified usage of Canvas Kit type token.

## Installation

```sh
yarn add @workday/canvas-kit-preview-react
```

## Text

### Usage

`Text` is the base component that is used to render text in the UI. It is built off of our `Box`
component which allows the use of styles props such as `display`, `margin`, `padding` as well as the
most common type styles such as `fontWeight`, `fontSize` amongst others. This component is an
extension of our type tokens(insert link), therefore, props like `variant`, `level` and `size` all
exists.

By default `Text` component is a `span` element, but it's possible to override the element by
passing `as` prop.

```tsx
<Text as="p" fontSize={14} fontWeight="medium">
  Text here...
</Text>
```

## Label

`Label` is a `label` element build based on the top of `Text` component with added ability to add
pointer cursor or disabled style.

### Usage

```tsx
<Label hasPointerCursor disabled={true}>
  Text here...
</Label>
```

## Type Level Components

Type Level Components, `TypeSubtextLevel`, `TypeBodyLevel`, `TypeHeadingLevel` and `TypeTitleLevel`,
apply type token styles to text element. Instead of passing type level token you can use one of Type
Level Components, like in example below:

```tsx
// v7
<p css={{...type.levels.subtext.small}}>Small Subtext Text</p>

// v8
<TypeSubtextLevel size="small">Small Subtext Text</TypeSubtextLevel>
```
