OVERVIEW
================================================================================

This is Canvas Kit version 14.0 upgrade guide for LLM assistance. It contains breaking changes, deprecations, removals, and migration instructions for upgrading from v13 to v14.

Key areas of change:
- Caution naming (Alert â†’ Caution)

================================================================================
BREAKING CHANGES
================================================================================

1. CAUTION NAMING
   - Alert/Warning states renamed to Caution
   - Changes affect: ErrorType.Alert â†’ ErrorType.Caution
   - Changes affect: error="alert" â†’ error="caution"
   - Impacts: FormField, TextInput, Switch, TextArea components
   - Codemod handles: YES

2. CANVAS PROVIDER (MAJOR BREAKING CHANGE)
   - Removed default branding color generation
   - Changed from chroma.js to oklch color generation
   - Theme prop updated to include 'lighter' property
   - May cause color discrepancies
   - Required: Provide full palette or use createCanvasTheme for chroma.js behavior

3. CARD COMPONENT
   - Default shadow removed (visual breaking change)
   - New variants: borderless, filled
   - Gap replaces individual margins (visual breaking change)
   - Uses display: flex with flex-direction: column

4. BUTTONS - TERTIARY BUTTON
   - Removed isThemable prop
   - Shape changed to rounded corners
   - Use cs prop or colors prop for customization

5. SEARCH FORM
   - SearchThemeAttributes: boxShadow and boxShadowFocus only accept string (not string array)
   - SearchTheme enum uses string values: 'light', 'dark', 'transparent'

6. STATUS INDICATOR (PREVIEW)
   - Variant name changes:
     * blue â†’ primary
     * green â†’ success  
     * orange â†’ caution
     * red â†’ critical
     * gray â†’ neutral
   - StatusIndicatorVariant type deprecated

================================================================================
REMOVALS
================================================================================

REMOVED COMPONENTS/UTILITIES:
1. DeprecatedButton - Use PrimaryButton, SecondaryButton, or TertiaryButton
2. InputProvider - No longer needed, use :focus-visible CSS
3. Menu (Preview) - Use Menu from Main package
4. readyOnlyPillStencil and removeablePillStencil - Use pillStencil
5. TextArea (Preview) - Use TextArea from Main package  
6. TextInput (Preview) - Use TextInput from Main package

================================================================================
MIGRATION EXAMPLES
================================================================================

CAUTION NAMING:
Before: <FormField error="alert">
After:  <FormField error="caution">

CANVAS PROVIDER:
Before: <CanvasProvider theme={{canvas: {palette: {primary: {main: 'purple'}}}}}>
After:  <CanvasProvider theme={{canvas: {palette: {primary: {main: cssVar(base.blue600)}}}}}>

CARD:
Before: <Card> (had shadow)
After:  <Card cs={{boxShadow: system.depth[1]}}> (for shadow)

AVATAR:
Before: <Avatar altText="John Doe" size="large" />
After:  <Avatar name="John Doe" size="large" variant="blue" />

STATUS INDICATOR:
Before: <StatusIndicator variant="blue" />
After:  <StatusIndicator variant="primary" />

PILL:
Before: <Pill variant="default"><Pill.Avatar altText="John Doe" /></Pill>
After:  <Pill><Pill.Avatar name="John Doe" /></Pill>
