# KBD

`KBD` is a compound component used to visually represent keyboard input, such as keys or keyboard
shortcuts.

## Installation

```sh
yarn add @workday/canvas-kit-labs-react
```

## Usage

### Basic Example

`KBD` is a container that wraps one or more `KBD.Item` components. Each `KBD.Item` represents a
single keyboard key.

<ExampleCodeBlock code={Basic} />

### RTL Example

You can also use the `KBD` component in right-to-left (RTL) layouts. This is helpful for languages
that are read from right to left.
