## Components

### Expandable

#### Usage

`Expandable` is a container component that holds a `Expandable.Target` and a `Expandable.Content`.
It provides a React Context model for its sub components.

#### Props

<!-- <ArgsTable of={Expandable} /> -->

### Expandable.Target

#### Usage

`Expandable.Target` creates a heading and a button. The button comes with the `aria-expanded` and
`aria-controls` attributes. When a user interacts with the button, they can show or hide the
`Expandable.Content`. If a string is passed to the Target as a child, it will be wrapped in a Title.

### Expandable.StartIcon

#### Usage

`Expandable.StartIcon` creates an icon, by default it uses a chevron. It will orient the icon
towards the title when the container is closed.

<!-- <ArgsTable of={Expandable.StartChevron} /> -->

### Expandable.EndIcon

#### Usage

`Expandable.EndIcon` creates an icon, by default is uses a chevron that points up. It will orient
the icon up when the container is closed.

<!-- <ArgsTable of={Expandable.EndChevron} /> -->

### Expandable.Content

#### Usage

`Expandable.Content` holds the content that will be conditionally opened and closed. It has an `id`
to ensure the `Expandable.Target` can properly set the `aira-controls` attribute.
